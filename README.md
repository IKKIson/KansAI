# KansAI

## setup

```
1. python3 설치하기
2. python3 가상환경 모듈 설치하기
    % python3-venv or % virtualenv
3. 원하는 파이썬 프로젝트 폴더에 가상환경 생성하기
    % python3 -m venv (가상환경 이름)
4. 가상환경 활성화
   -> (1) win10
      %  ./venv/Scripts/activate
      (2) linux
      %  source venv/bin/activate
5. 가상환경에 모듈 설치하기
   -> pip install [모듈명] [모듈명] [모듈명==버전]
      pip install numpy matplotlib pandas
6. 가상환경이 설치된 모듈 삭제하기
   pip uninstall [모듈명]
7. 가상환경에 설치된 모듈들 목록 보기
   pip list
8. 가상환경에 설치된 모듈 백업하기
   pip freeze > requirements.txt
9. requirement.txt의 모듈목록 설치하기
   pip install -r requirements.txt
10. 가상환경 비활성화 (윈도우, 리눅스 동일)
   deactivate
 !!!주의사항!!!
 !!! 설치된 python3의 실행 명령이 python인지 python3인지 구분확인!!!
 모듈 설치 시 pip 써야되는지 pip3 써야되는지 구분
```
